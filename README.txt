Despliegue:
Poryecto optmizado para exportar:
- Abre el proyecto con Godot Engine.
- Corre la escena principal inicio.scn
- Exporta el videojuego en Android:
  Necesitarás complir con estos requisitos: https://docs.godotengine.org/en/stable/tutorials/export/exporting_for_android.html
- Preparado para exportar en Windows.
